# Thesis Manuscript

Main file:

- `master_thesis.tex`

Build output:

- `master_thesis.pdf`

Chapter files:

- `chapters/chapter1_introduction.tex`
- `chapters/chapter2_background_related_work.tex`
- `chapters/chapter3_methodology.tex`

Bibliography:

- `references.bib`

Open `master_thesis.tex` in VS Code and run the LaTeX build command. You can also build while editing a file under `chapters/` because each chapter points back to the main root file.
